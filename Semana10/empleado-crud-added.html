Aplicación Web CRUD con Filtrado y Paginación

1. Descripción de la Aplicación

En este ejercicio, agregaremos funcionalidades para mostrar los datos completos del empleado en un card, filtrar datos de los empleados y agregar una barra de paginación que permita mostrar 8 empleados por página.

2. Procedimiento

  1. Mostrar datos completos del empleado: Agregar un botón que, al ser clickeado, muestre los datos completos del empleado en un card.
  2. Filtrar datos de los empleados: Implementar un input de texto y un botón para filtrar los empleados según el criterio ingresado.
  3. Agregar barra de paginación: Implementar una barra de paginación que permita mostrar 8 empleados por página y navegar entre las páginas.
