Desarrollo de Aplicaciones Dinámicas con React usando Hooks

Hooks son funciones de JavaScript que permiten crear y acceder al estado y al ciclo de vida del componente en React.

Lista de Hooks importantes:

  • useState: Permite agregar estado local a componentes funcionales.
  • useEffect: Permite realizar efectos secundarios en componentes funcionales.
  • useContext: Permite acceder a un contexto en un componente funcional.
  • useReducer: Maneja el estado complejo con lógica de reducción.
  • useRef: Permite crear referencias mutables que persisten entre renderizados.
  • useNavigate: Permite la navegación programática entre rutas en React Router.
  • useParams: Accede a los parámetros de la URL en React Router.

Gestión de Estados en React

La gestión de estados es el proceso de seguimiento y actualización del estado de una aplicación. El estado es la condición actual de una aplicación, como la ubicación del usuario, los artículos en su carrito de compras o el estado de un juego.

El estado de un componente es un objeto que contiene los datos (variables) que necesitamos para representarla en la interfaz u otro contexto.

Ejemplos de Hooks en React

useState: Permite agregar estado local a componentes funcionales.


                    import React, { useState } from 'react';

                    function Contador() {
                        const [contador, setContador] = useState(0);

                        return (
                            <div>
                                <p>Contador: {contador}</p>
                                <button onClick={() => setContador(contador + 1)}>Incrementar</button>
                            </div>
                        );
                    }

useEffect: Permite realizar efectos secundarios en componentes funcionales.


                    import React, { useState, useEffect } from 'react';

                    function Temporizador() {
                        const [segundos, setSegundos] = useState(0);

                        useEffect(() => {
                            const intervalo = setInterval(() => {
                                setSegundos(segundos => segundos + 1);
                            }, 1000);

                            return () => clearInterval(intervalo);
                        }, []);

                        return <p>Segundos: {segundos}</p>;
                    }

useContext: Permite acceder a un contexto en un componente funcional.


                    import React, { useContext } from 'react';

                    const TemaContext = React.createContext('light');

                    function MostrarTema() {
                        const tema = useContext(TemaContext);
                        return <div>Tema actual: {tema}</div>;
                    }

¿Qué aprendí?

Aprendí a utilizar diferentes Hooks de React para gestionar el estado y los efectos secundarios en componentes funcionales. Los Hooks me permiten escribir componentes más limpios y concisos sin necesidad de convertirlos en componentes de clase.

¿Cómo aprendí?

Lo aprendí a través de la práctica, creando ejemplos sencillos que implementan cada uno de los Hooks importantes y observando cómo simplifican el código comparado con la gestión de estado y ciclo de vida en componentes de clase.
