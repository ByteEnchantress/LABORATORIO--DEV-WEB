Desarrollo de una Aplicación Web con Hooks Personalizados y useEffect

Desarrollar una aplicación web que utilice hooks personalizados y useEffect para crear un widget (150px x 150px) con una imagen(5 imagenes) que cambia aleatoriamente cada 30 segundos.

2. Procedimiento

    1. Crear el Hook Personalizado: Este hook manejará la lógica del cambio de imagen usando el hook useEffect de React para actualizar la imagen cada 30 segundos.
    2. Definir las Imágenes:Seleccionar las 5 imágenes que se utilizarán en el widget.

      Imágenes ilustrativas del widget en funcionamiento:

    3. Implementar el Componente del Widget: Crear un componente que utilice el hook personalizado para mostrar la imagen aleatoria.
    4. Integrar el Componente en la Aplicación: Añadir el componente del widget a la aplicación principal y probar su funcionamiento.

3. Código de Implementación

import { useEffect, useState } from 'react'; import image1 from '../assets/img1.jpg'; import image2 from '../assets/img2.jpg'; import image3 from '../assets/img3.jpg'; import image4 from '../assets/img4.jpg'; import image5 from '../assets/img5.jpg'; const images = [image1, image2, image3, image4, image5]; const useRandomImage = () => { const [image, setImage] = useState(images[0]); const [fadeClass, setFadeClass] = useState(''); useEffect(() => { const interval = setInterval(() => { setFadeClass('fade-out'); setTimeout(() => { const randomIndex = Math.floor(Math.random() * images.length); setImage(images[randomIndex]); setFadeClass(''); }, 500); }, 3000); return () => clearInterval(interval); }, []); return { image, fadeClass }; };
