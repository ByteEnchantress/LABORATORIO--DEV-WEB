Más sobre React + Vite

React es una biblioteca de JavaScript para construir interfaces de usuario. Facilita la creación de aplicaciones web interactivas y dinámicas mediante el uso de componentes reutilizables y un modelo de datos unidireccional.

Vite es una herramienta de desarrollo que mejora la experiencia de desarrollo de aplicaciones web. Proporciona una configuración mínima y un servidor de desarrollo rápido, así como un sistema de construcción optimizado.

Ejemplo de React

Con React, puedes crear componentes de interfaz de usuario de manera declarativa. Por ejemplo:

  • <button onClick={() => alert('Clicked!')}>¡Haz clic!</button>
  • <h1>{title}</h1> donde title es una variable de estado.

Ejemplo de Vite

Con Vite, puedes iniciar un nuevo proyecto rápidamente usando un comando como:

  • npm init vite@latest my-app --template react
  • Y luego, ejecutando npm run dev para iniciar el servidor de desarrollo.

Ejercicio: Crear un Proyecto con React + Vite

Para este ejercicio, seguí estos pasos para crear un proyecto usando React con Vite:

  1. Inicié un nuevo proyecto con el siguiente comando:
    2. npm create vite@latest mi-app --template react
  2. Navegué al directorio del proyecto:
    3. cd mi-app
  3. Instalé las dependencias:
    4. npm install
  4. Inicié el servidor de desarrollo:
    5. npm run dev
