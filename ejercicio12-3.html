Array Multidimensional JSP

Visualización de datos de estudiantes en una tabla estilizada

1. Descripción del Proyecto

Declarar un array multidimensional con los datos de 10 estudiantes y mostrar estos datos en una tabla estilizada en un archivo JSP.

2. Procedimiento

  1. Declaración del Array Multidimensional

    Definir el array multidimensional en un archivo JSP.

  2. Creación de la Tabla

    Mostrar los datos del array en una tabla HTML estilizada con CSS.

3. Código de Implementación

<% String[][] estudiantes = { {"1", "Luis", "Padilla", "20"}, {"2", "Lilo", "Galarza", "22"}, {"3", "Polo", "Lopez", "21"}, {"4", "Maria", "Rodríguez", "23"}, {"5", "Carlos", "Sánchez", "24"}, {"6", "Lucía", "Fernández", "20"}, {"7", "Pedro", "Gómez", "22"}, {"8", "María", "López", "21"}, {"9", "Jesus", "Díaz", "23"}, {"10", "Sofía", "Ramírez", "24"} }; for (int i = 0; i < estudiantes.length; i++) { out.println(""); for (int j = 0; j < estudiantes[i].length; j++) { out.println("" + estudiantes[i][j] + ""); } out.println(""); } %>
